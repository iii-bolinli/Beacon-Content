## Android Beacon 推播、電量資訊查詢 SDK
### 資策會創研所
> ####檔案說明

* 主要檔案為：BeaconContentSDK、BeaconContentDemo

* BeaconContentSDK 為 SDK 本體，分為包含 Gson (master) 檔案以及不包含 Gson (No_Gson) 檔案兩種，避免使用時原專案中已經有 Gson 檔案造成衝突

* 範例程式 BeaconContentDemo 建議使用 Eclipse 開啟
